🔗 Google Drive 연동 설정
Google Cloud Console에서 OAuth 클라이언트 ID를 발급받아 입력하세요. 아래 안내를 따라 5분이면 완료됩니다.
Step 1
Google Cloud Console 접속
console.cloud.google.com 에 접속 후 프로젝트를 생성(또는 선택)하세요.
Step 2
Google Drive API 활성화
좌측 메뉴 → API 및 서비스 → 라이브러리 → "Google Drive API" 검색 → 사용 설정
Step 3
OAuth 동의 화면 설정
API 및 서비스 → OAuth 동의 화면 → 외부 선택 → 앱 이름 입력 → 저장
테스트 사용자에 본인 Gmail 추가
Step 4
OAuth 클라이언트 ID 생성
사용자 인증 정보 → 사용자 인증 정보 만들기 → OAuth 클라이언트 ID
유형: 웹 애플리케이션
승인된 JavaScript 출처:
생성 후 클라이언트 ID를 복사하세요.
